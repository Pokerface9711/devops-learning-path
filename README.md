## 💡 欢实的建议

1. **别光看视频**，手必须脏。每学一个东西都自己搭一遍。
2. **别追新**，先把基础工具用熟。
3. **别孤立学**，学的时候想想"这玩意儿和之前的怎么配合"。
4. **出了问题先看日志**，90% 的问题日志里都写着答案。
