## 📝 检查清单

在进入下一阶段前，确保你能做到：

### Linux
- [ ] 能用 `ls`、`cd`、`mkdir`、`rm`、`cp`、`mv` 管理文件
- [ ] 理解文件权限（chmod、chown）
- [ ] 能查看进程（ps、top、htop）和杀进程（kill）
- [ ] 会用 `ssh` 连接服务器
- [ ] 会用 `vim` 或 `nano` 编辑文件
- [ ] 能写基础 bash 脚本（变量、循环、条件）

### 网络
- [ ] 理解 IP 地址、子网掩码、网关
- [ ] 知道 DNS 是怎么工作的
- [ ] 理解 HTTP 请求/响应结构
- [ ] 能用 `curl` 测试 API
- [ ] 能配置简单的防火墙规则

### Git
- [ ] 能 clone、add、commit、push、pull
- [ ] 理解分支（branch）和合并（merge）
- [ ] 能解决简单的冲突
- [ ] 知道 rebase 和 cherry-pick 是什么
